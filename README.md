# Fundamental C Programming & Conditionals

A structured repository containing foundational C programming exercises, progressing from basic arithmetic operations to conditional logic structures. This repository serves as a core programming foundation for academic excellence and algorithmic problem-solving.

## 📌 Overview

This repository is divided into two primary modules:
1. **Fundamental C Programming:** Basic syntax, data types, input/output operations, and mathematical formulas.
2. **Conditionals:** Decision-making structures (`if`, `if-else`, `else if`) to handle logical branching.

---

## 📂 Repository Structure

The files are organized sequentially based on topic complexity:

### 🧩 Fundamental C Programming
* `1. Sum of Two Integer.c` - Basic user input and addition.
* `2. Swap Two Numbers.c` - Variable manipulation using temporary variables.
* `3. Area and Perimeter of a Rectangle.c` - Implementation of geometric formulas.
* `4. Arithmetic Operations on Two Integers.c` - Handling addition, subtraction, multiplication, and division.
* `5. Print Greeting.c` - Standard I/O operations.
* `6. Calculate Area and Circumference of a Circle.c` - Working with floating-point numbers and constants ($\pi$).
* `7. Calculate the Area of a Triangle.c` - Implementing standard geometric computation.
* `8. Calculate A Raised to the Power B.c` - Using basic loops or the `<math.h>` library.
* `9. Calculate Compound Interest.c` - Advanced mathematical formula implementation.
* `10. Convert Hours to Minutes.c` - Unit conversion logic.

### 🔀 Conditionals
* `11. Even or Odd.c` - Modular arithmetic (`%`) for parity checking.
* `12. Leap Year.c` - Complex logical conditions using `&&` and `||`.
* `13. Weekdays.c` - Multi-way branching or `switch-case` structures.
* `14. Valid Voter.c` - Relational operators for age verification.
* `15. Vowel or Consonant.c` - Character characterization and logical grouping.
* `16. Two Numbers Are Equal.c` - Equality testing between variables.
